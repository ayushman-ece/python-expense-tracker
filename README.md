# 💰 Python Expense Tracker

A simple Command Line Expense Tracker built using Python.

This project allows users to manage their daily expenses by adding, viewing, and deleting expense records. All data is stored permanently using a JSON file.

---

## 🚀 Features

- ➕ Add Expense
- 📋 View All Expenses
- ❌ Delete Expense
- 💾 Store Data in JSON File
- 📂 Persistent Data Storage

---

## 🛠 Technologies Used

- Python 3
- JSON
- File Handling
- OS Module
